# Welcome to My Digital Brain

This is my personal knowledge base where I document everything I learn, explore, and discover.

## How to Use This Site

- Use the **navigation sidebar** to browse topics - it automatically organizes all your folders and files
- Use the **search bar** at the top to find specific content quickly
- Click the **theme toggle** to switch between light and dark modes

## Getting Started

Simply create new folders and markdown files in the `docs/` directory. The navigation will automatically update to reflect your structure.

### Example Structure

```
docs/
├── programming/
│   ├── python.md
│   └── javascript.md
├── tools/
│   └── git.md
└── notes/
    └── daily-notes.md
```

Each folder and file you create will automatically appear in the sidebar navigation.

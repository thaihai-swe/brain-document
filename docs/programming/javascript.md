# JavaScript

JavaScript is a versatile programming language primarily used for web development.

## Overview

JavaScript enables interactive web pages and is an essential part of web applications.

## Example

```javascript
function greet(name) {
    return `Hello, ${name}!`;
}

console.log(greet("World"));
```

## Modern JavaScript

- ES6+ features
- Async/await
- Arrow functions
- Destructuring
